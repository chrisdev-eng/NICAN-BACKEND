# NICAN-BACKEND

PROJETO-INTEGRADOR 

Alunos: Herinque Pantaleão, Christian Ferreira, Daniel Nunez

O backend do site Nican Mopohua é um sistema de gestão para grupos escoteiros, desenvolvido para digitalizar e otimizar processos administrativos internos. Ele centraliza a gestão de presenças e materiais em uma plataforma segura, com controle de acesso por perfis, permitindo que jovens e chefes utilizem funcionalidades específicas conforme seu nível hierárquico. Esta implementação serve como base para a futura evolução do projeto em uma aplicação web completa.
